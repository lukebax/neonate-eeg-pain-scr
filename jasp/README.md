
# Raincloud Plots (JASP)

This folder contains pre-configured JASP files used to generate raincloud plots for the neonatal EEG pain scoping review. The plots summarize postmenstrual age, sex distribution, sample size, and EEG data loss.

## Folder contents

- `raincloud_data_single.jasp`: Generates plots for EEG sample size and data loss
- `raincloud_data_paired.jasp`: Generates plots for postmenstrual age and sex

## How to use this folder

1. Run the analysis script `r/scripts/analyse_data.R` (see the `r/` folder). This creates the necessary CSV files:
   - `raincloud_data_single.csv`
   - `raincloud_data_paired.csv`

2. Open either `.jasp` file in *JASP*. The corresponding CSV file is already loaded, and all raincloud plot settings are pre-configured.

   - The raincloud plots are located under the **Descriptives** tab.

## Output files and usage

- The `.jasp` files visualize the cleaned CSV data exported from the R analysis.
- These plots can be viewed, customized, or exported for use in publications.

## Summary

These pre-set *JASP* files allow immediate visualization of the raincloud data generated by the R analysis. No manual configuration is required.
